# DrinkMe WEB Client

DrinkMe Web Client

## How to install

```$
$ git clone https://github.com/yapp-project/team02_frontend.git
$ cd DrinkMe
$ npm install
```

## How to run

```$
$ npm start
```

and enter to `localhost:3000`

## Skill Stacks

- React v16
- React-Router v4
- React-Redux, Redux-Thunk
- Sass
